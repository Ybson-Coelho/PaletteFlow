# PaletteFlow

PaletteFlow is a modern color palette generator built with HTML, Tailwind CSS, and JavaScript.

Generate beautiful palettes instantly from curated color styles like pastel, neon and dark. Built to be fast, responsive, and simple.

## ✨ Features

* Generate random color palettes
* Multiple color styles
* Copy HEX colors instantly
* Responsive design
* Keyboard shortcut support (`Space`)
* Smooth animations and transitions
* Clean and modern UI

## 🚀 Technologies

* HTML5
* Tailwind CSS
* JavaScript (ESModules)

## 📁 Project Structure

```txt
project/
│── index.html
│── script.js
└── palettes/
    │── pastel.js
    │── neon.js
    │── dark.js
```

## ⚡ Getting Started

Clone the repository:

```bash
git clone <repository-url>
```

Open the project with a local server.

If you're using VSCode:

1. Install the `Live Server` extension
2. Right click `index.html`
3. Click `Open with Live Server`

## 🎨 Available Styles

* Pastel
* Neon
* Dark

## ⌨️ Shortcut

Press:

```txt
Space
```

to generate a new palette instantly.
